The aim of this repository is to leverage Michael Nielsen's neural network source code for the classification of images from the Fashion Mnist dataset. 
Throughout this undertaking, my primary goal is to mitigate overfitting while fine-tuning the hyperparameters to achieve optimal performance.
I intend to make full use of Michael Nielsen's comprehensive code repository, which not only contains the neural network implementation but also features 
a remarkable graphing program tailored for overfitting analysis.
